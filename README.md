# 📚 Data Structures and Algorithms in C

This repository contains the C programs developed for the **Data Structures and Algorithms (DSA)** course. The programs are organized unit-wise according to the course assignment and include additional implementations for better understanding.

---

## 🛠️ Language Used

- C Programming
- GCC Compiler

# 🔐 Hashing Implementations

The **Hashing** folder contains implementations of various hashing techniques and collision resolution methods.

### Hash Functions

- Division Method
- Folding Method
- Mid-Square Method
- Multiplication Method
- DJB2 Hash Function
- Universal Hashing
- Perfect Hashing
- Rehashing
- Cryptographic Hash Function Simulation

### Collision Resolution

- Linear Probing
- Quadratic Probing
- Double Hashing
- Separate Chaining
